CVPR 2026 Highlight

Training One Model to Master Cross-Level Agentic Actions via Reinforcement Learning

A CraftJarvis project for cross-level agentic actions in Minecraft.

Kaichen He, Zihao Wang, Muyao Li, Anji Liu, Yitao Liang

Paper Code

Overview

CrossHA studies how to train one agentic model with reinforcement learning to handle cross-level actions in Minecraft. The released implementation is part of the OpenHA codebase, with the CrossAgent folder containing the related training code.

Highlights

One Model

A unified policy is trained for cross-level agentic actions.

Reinforcement Learning

The work focuses on RL-based training for embodied Minecraft agents.

OpenHA Release

Code is released through the CraftJarvis OpenHA repository.
